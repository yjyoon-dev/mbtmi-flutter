# 엠비티엠아이

엠비티엠아이(MBTMI)는 **2022 SW중심대학 공동해커톤**에서 **우수상**을 수상한 프로젝트로, 과몰입 팀이 개발한 MBTI 익명 커뮤니티 플랫폼 서비스 입니다. MBTI 별 익명 게시판 기능을 지원하며, MBTI 분석 서비스 및 다양한 컨텐츠들을 한 번에 모아볼 수 있습니다.
